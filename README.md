# Emulador de Paginacao em C

Este projeto simula a traducao de enderecos virtuais para fisicos utilizando tres estruturas de paginacao:

- Paginacao hierarquica
- Paginacao por hash
- Tabela de paginas invertida

## Como rodar o projeto

### Pre-requisitos

Antes de executar, e necessario ter um compilador C instalado, como o GCC.

Para verificar se o GCC esta instalado, execute:

```bash
gcc --version


gcc teste.c -o teste.exe


./teste.exe
